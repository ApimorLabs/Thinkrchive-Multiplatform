Project Details

Thinkrchive - Multiplatform is an Android & Desktop app that shows specifications and details of various Lenovo Thinkpad models. This project was made to try out Jetpack Compose and Kotlin Multiplatform so that I can see how I could adopt it to my own personal apps.

This app will also be documented heavily so that it can help anyone else in the future that is trying to get into Kotlin Multiplatform to support Android and Desktop via Compose Multiplatform

This project is a Mutliplatform version of the initial Thinkrchive which was Android only
